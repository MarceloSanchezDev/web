Proyectos

E-COMERCE

Desarrollé este e-commerce utilizando HTML, CSS y JavaScript para crear una experiencia de compra en línea intuitiva y atractiva. Este proyecto muestra mi habilidad para crear una interfaz fácil de usar, permitiendo a los usuarios explorar productos, agregar artículos al carrito de compras y realizar pagos seguros de manera fluida. Destacando un diseño responsivo y funcionalidades como filtros de búsqueda, páginas detalladas de productos y un proceso de pago seguro, este e-commerce es un ejemplo de mi enfoque en la creación de soluciones web efectivas y atractivas para los usuarios finales

APP DE CLIMA

Desarrollé una aplicación de clima utilizando React y la API de OpenWeatherMap para proporcionar a los usuarios información actualizada y precisa sobre el clima. Esta aplicación permite a los usuarios buscar ciudades y obtener datos meteorológicos detallados, como la temperatura actual, la previsión extendida, la humedad y la velocidad del viento. Aprovechando React, esta app ofrece una interfaz interactiva y receptiva, permitiendo una experiencia de usuario fluida al mostrar datos meteorológicos relevantes de manera clara y concisa. Este proyecto demuestra mi capacidad para integrar API externas, trabajar con React para crear interfaces dinámicas y proporcionar una experiencia útil y práctica para los usuarios interesados en conocer el clima de distintas ubicaciones
